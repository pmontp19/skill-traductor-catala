# traductor-catala

Skill per a Claude Code que ajuda a traduir programari al català seguint els estàndards de [Softcatalà](https://www.softcatala.org).

## Instal·lació

```bash
npx skills add pmontpeo/traductor-catala
```

O manualment: copia la carpeta `traductor-catala/` a `~/.claude/skills/`.

## Contingut

La skill aplica automàticament quan estàs traduint cadenes de programari al català:

- **Guia d'estil de Softcatalà** — registre, verbs d'UI, puntuació, majúscules, barbarismes
- **Normes de localització** — formats de data/hora/número, locale codes, unitats SI
- **Convencions de format** — guillemets catalanes, guions, sigles, tecles
- **Terminologia** — glossari Softcatalà, TERMCAT, 80+ termes UI recomanats, accés a memòries de traducció TMX

## Fonts

- [Guia d'estil de Softcatalà](https://www.softcatala.org/guia-estil-de-softcatala/)
- [Memòries de traducció](https://www.softcatala.org/recursos/memories/)
- [Recursos terminològics](https://www.softcatala.org/recursos/terminologia/)
- [Estàndards ISO en català](https://www.softcatala.org/estandard-iso-catala/)
